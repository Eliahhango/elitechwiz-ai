## ⚡ **3-Step Quick Start**

### 1️⃣ **Get Your API Key**
- 🔑 **OpenAI**: [Get API Key](https://platform.openai.com/api-keys) (Recommended)
- 🔑 **Anthropic**: [Get API Key](https://console.anthropic.com/) (Alternative)

### 2️⃣ **Deploy Instantly**
- Click any deployment button above
- Paste your API key when prompted
- Wait 2-3 minutes for deployment

### 3️⃣ **Connect WhatsApp**
- Visit your deployed URL
- Follow the setup wizard
- Scan QR code or use pairing code
- ✅ **Done!** Your AI bot is live!

## 🛠️ **Local Development**

<details>
<summary><strong>💻 Development Setup</strong></summary>

\`\`\`bash
# 1️⃣ Clone the repository
git clone https://github.com/Eliahhango/elitechwiz-ai.git
cd elitechwiz-ai

# 2️⃣ Install dependencies
npm install

# 3️⃣ Configure environment
cp .env.example .env
# Edit .env with your API keys

# 4️⃣ Start development server
npm run dev

# 5️⃣ Open browser and follow setup wizard
# http://localhost:3000
\`\`\`

</details>

<details>
<summary><strong>🐳 Docker Deployment</strong></summary>

\`\`\`bash
# Quick Docker setup
git clone https://github.com/Eliahhango/elitechwiz-ai.git
cd elitechwiz-ai

# Create environment file
cp .env.example .env
# Add your API keys to .env

# Build and run
docker build -t elitechwiz-ai .
docker run -d --name elitechwiz-ai \
  -p 3000:3000 \
  -v $(pwd)/baileys_auth_info:/app/baileys_auth_info \
  --env-file .env \
  elitechwiz-ai

# Check logs for pairing code
docker logs elitechwiz-ai
\`\`\`

</details>

OPENAI_API_KEY=sk-your-openai-key-here

## 🐛 **Troubleshooting**

<details>
<summary><strong>❌ Common Issues & Solutions</strong></summary>

### 🔐 **Authentication Issues**
**Problem**: QR code not showing
- **Solution**: Use pairing code method instead
- Set `USE_PAIRING_CODE=true` in environment

**Problem**: Pairing code not working
- **Solution**: Ensure phone number includes country code
- Format: `+1234567890` (with + and country code)

### 🌐 **Connection Issues**
**Problem**: Bot disconnects frequently
- **Solution**: Check internet stability
- Review error logs for specific issues
- Ensure proper session persistence

**Problem**: Messages not being received
- **Solution**: Verify bot is activated with `/start`
- Check if WhatsApp Web is connected
- Restart the bot if necessary

### 🤖 **AI Response Issues**
**Problem**: "API key not found" error
- **Solution**: Verify API key is correctly set
- Check environment variable spelling
- Ensure API key has sufficient credits

**Problem**: Slow AI responses
- **Solution**: Switch to faster model (gpt-3.5-turbo)
- Check API quota and usage limits
- Consider upgrading API plan

</details>
